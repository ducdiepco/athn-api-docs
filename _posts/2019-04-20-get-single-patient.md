Returns a particular patient

### Path
```GET /patient/:id ```

### Parameters
```id: integer - Patient id ```

### Response Example

``` {
  "id" : 1001,
  "first_name" : "Wallas",
  "last_name" : "Rydzynski",
  "email" : "wrydzynski0@blinklist.com"
} ```
