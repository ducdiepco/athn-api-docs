Create a new patient

### Path
```POST /patient ```

### Parameters

```first_name: String - first name of patient ```
```last_name: String - last_name of patient```
```email: String - email of patient```

### Response
Return the new patient ID number

### Response Example
```{
  "id" : 1200
}```
